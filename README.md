- Easily navigate between sections using the navigation menu at the top.

## Installation Instructions
To set up this project locally, follow these steps:

1. **Clone the repository**:
   Open your terminal and run the following command to clone the repository to your local machine:
   ```bash
   git clone https://github.com/JinxYourIt/JinxYourIt.github.io.git
   ```
2. **Change to project directory**:
   In terminal, run the following command to switch to the project directory:
   ```bash
   cd JinxYourIt.github.io
   ```
3. **Open project**:
   Open index.html file in your preferred browser.
